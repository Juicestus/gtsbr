NOTES

351 What is an internet service provider?

A company that provides access from a private network to the internet of from an individual’s home computer to the internet.

353 How is the internet structured?

By an interconnected hierarchy of ISPs.

354 What is the internet?

The huge pictorial representation of the interconnected networks that make up the internet.

354 What is a firewall?

Hardware and software placed at the entry points to a private network to regulate traffic.

355 How many nodes are on the internet?

Over a billion independent machines.

356 What is a communication protocol?

A protocol is a mutually agreed upon set of rules, conventions, and agreements for the efficient and orderly exchange of information.

356 What is the Internet Society?

A nonprofit, nongovernment professional society of 145 worldwide organizations  in 100 countries.

357 What is the physical layer protocol?

A protocol to govern the exchange of the binary digits and bits across a network.

357 What is the goal of the physical layer?

To create a bit pipe using voltage levels, wavelengths, and radio frequency.

358 What is the data link layer?

The rules to govern the structure of data into bits, and error correction.

358 What is error detection and correction?

Detecting the differences between the input and output bits in a connection.

358 What is framing?

Identifying the start and stop of a message or data set.

358 What are Medium Access Control protocols?

Controls ownership of a shared line.

360 What is a logical link layer?

Ensures a message travels safely from source to destination.

360 What is an ARQ algorithm?

Automatic Repeat Requests, the basis for all Data Link Control protocols. Retransmits a new copy of the data if not received by the receiver.

361 What is and Acknowledgment Message?

(ACK) Message to say that data has been received correctly.

363 What is Internet Protocol?

The network layer (glue) on the internet.

363 What is a host name?

The symbolic names of the hosts of a webpage.

363 What is an IP address?

The 32-bit address of a host on the internet.

363 What is Domain Name System?

(DNS) A system to convert a symbolic host name to an IP address.

368  What is the Transport Layer?

The layer of protocols to direct signals to individual ports.

368 What is a port number?

An integer value to represent an individual program on a host computer.

369 What is TCP?

Transport Layer Protocol. The current transport protocol in use.

371 What is the application layer?

Rules for implementing end to end user services in individual programs
