395 What is the purpose of Authorization and Authentication?

To be the first line of defense in preventing the illicit use of or threats to a computer resource or sensitive information.

395 What is authentication?

Authentication is verifying who a user is. When you log into a computer or online account with a password, you are being authenticated as the owner of the account or computer user.

396 What is the proper definition of authentication?

The process of verifying that you really are the person who is accessing a resource or information.

396 What is a hacker?

A person who knows how to (and does) gain unauthorized access to a computer with malicious intent. Hacking is illegal and punishable under the law with fines and imprisonment.

396 What is cyberwarfare?

Politically motivated information theft or hardware damage for porpoise of sabotage and/or espionage.

397 What is encryption?

Using algorithms to obfuscated passwords.

397 Why use encryption?

To prevent file breaches from leaking passwords.

397 What algorithms are used to do this?

A hash function.

397 What is a hash function?

A one way function that easy to apply and hard to undo, used to salt and hash passwords for storage.

398 What is a timestamp?

The current time an operation is performed at.

399 What is password cracking?

Trying to find passwords using educated guessing methods.

399 What is password cracking software?

Programs that crack passwords using certain methods.

399 What is a rainbow table?

A precomputed table of words and their hashes.

399 What is social engineering?

The process of using people to get the information you want.

399 What are security questions?

Questions of personal information to complement a password in a login.

399 What is biometric information?

Personal features like fingerprints, retina, and facial scans used for authentication.

400 How can you make your passwords better?

Use a password manager to allow you to use more secure passwords.

400 What is a password manager?

A vault to store passwords allowing you to not have to remember long and unique passwords.

401 What is authorization?

What governs what a user is allowed to do and what they have access to.

401 What is an access control list?

A list that specifies the privileges of certain users, such as:

  • Read access
  • Write access
  • Execute access
  • Delete access

402 What is a system administrator?

Aka superuser (sudo): a user with access to everything on the system.

402 What is malware?

Malicious software, a program that can attack an individual’s computer.

402 What is a virus?

A malware program that spreads similar to a virus, in that it sends itself to other computers from its host inside things like emails that the host user sends.

402 What is a worm?

Similar to a virus, a worm can transmit from node to node automatically and is a self-replicating software that does not need to infect the host.

403 What is spam?

Junk email.

403 What is a Trojan Horse?

A piece of software that pretends or is legitimate but downloads a malware payload on the host. A common payload is a keystroke logger.

403 What is a keystroke logger?

A piece of software that logs keystrokes in attempt to store passwords and other confidential data that the user typed.

403 What is a drive-by exploit of drive-by download?

Visiting an infected website downloads Trojan Horse malware onto a computer.

403 What is denial of service attack?

Aka DoS, a cyber-attack which directs a large number of HTTP requests from many different clients to a website to overload the server and shut it down.

404 What is distributed denial of service attack?

Aka DDoS, a cyber-attack which uses a zombie army or botnet of infected computers to launch a DoS attack on a server. Harder to combat because packets come from many different IP’s.

404 What is a zombie army or botnet?

Many infected computers under someone else’s control for use in malicious activity.

404 What is a reflection and amplification DDoS?

DDoS queries of a relatively small and normal size to the server, but that trigger the server to return a huge amount of data back. Much harder to combat because the client sends little information.

404 What is phishing?

Claiming to be an organization that would ask for information, and letting users give that information up under the false assumption that the site is legitimate.

406 What is antivirus software?

A piece of software from a reputable company to detect and combat worms, viruses, and Trojan Horses.

406 What is firewall software?

A piece of software from a reputable company to guard the access points to your computer over a network.

406 What is antispyware?

A piece of software from a reputable company to detect spyware programs.

406 What is spyware?

Software that captures information on websites, passwords, and card numbers you have used.

406 What is a pop-up blocker?

Software that blocks small “pop-up” windows from appearing, which many times contain downloads for malware.

406 What are security patches.

Software updates to your operating system designed to fix vulnerabilities in the operating system.

407 What are white hat hackers?

White hat hackers tryp to exploit vulnerabilities so they can be fixed before they can be exploited by a black hat hacker.

407 What are black hat hackers?

Black hat hackers try to exploit vulnerabilities with malicious intent.
